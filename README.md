## 팀 역할 분배
> **팀장:** 김민지
> 
> 담당업무 : 백엔드 및 디자인 담당
> 
> 담당파트 : 장바구니, 주문, 전체 리뷰 및 게시판 개발과 디자인
>

> **팀원:** 김태현
>
> 담당업무 : 전체적인 백엔드 총괄
>
> 담당파트 : 상품, 장바구니, 주문, 컨텐츠 페이지 백앤드 개발


> **팀원:** 이진희
>
> 담당업무 : 전체적인 디자인 총괄
> 
> 담당파트 : 루틴, 컨텐츠 페이지 기획 및 제작, 전체적인 페이지 개발과 디자인
> 

> **팀원:** 정현민
>
> 담당업무 : 회원
>
> 담당파트 : 회원가입, 마이페이지
